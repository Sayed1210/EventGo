# EventGo 🎟️

A full-featured event booking platform for concerts, cinemas, and sports.

## Technologies
- ASP.NET Core MVC
- C# / Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Stripe Payment API
- Bootstrap 5

## Features
- User Authentication (Register/Login/Logout)
- Role-based Authorization (Admin/User)
- Event Browsing & Booking
- Stripe Payment Integration
- Admin Dashboard
- Search & Filter by location/name

## Setup
1. Clone the repo
2. Update connection string in appsettings.json
3. Run migrations: `dotnet ef database update`
4. Add Stripe keys in appsettings.json
5. Run the project
